【要件・README】
# SES案件マッチング管理ツール

ReForce株式会社 SES営業部向けの案件・人材マッチング管理Webアプリ。

## メインURI

- `index.html` — SPA（シングルページアプリケーション）

## 実装済み機能

### ダッシュボード
- KPIカード（Open案件数・要員数・マッチング数・共有可能案件数）
- 最近追加された案件一覧
- 案件検索中の要員一覧

### 案件管理
- 案件一覧テーブル（ステータス・共有・追加日・案件名・職種・勤務地・働き方・仕入単価・案件元・提案件数）
- 検索・フィルタ（共有可否・働き方）
- CRUD（追加・編集・削除）
- 詳細モーダル（紐付き要員一覧付き）
- スキルタグ（必須/尚可）・業界タグ・必要経験年数・仕入単価(数値)

### 要員管理
- 要員カード表示（スキルタグ・提案中案件バッジ付き）
- 検索
- CRUD（追加・編集・削除）
- 詳細モーダル（紐付き案件一覧付き）
- スキルタグ・業界タグ・経験年数・希望単価(数値)

### マッチング
- 案件選択 → 自動候補提示（スコア順）
- マッチング登録・編集・削除
- マッチング履歴テーブル

### マッチングロジック（v3）
| 要素 | 配点 |
|---|---|
| 必須スキルタグ完全一致 | +3点/件 |
| 必須スキル類似一致 | +1.5点/件 |
| 尚可スキルタグ完全一致 | +1点/件 |
| 尚可スキル類似一致 | +0.5点/件 |
| 業界タグ一致 | +2点/件 |
| ポジション一致 | +2点 |
| 経験年数充足 | +1点 / 不足時 -1点 |
| 単価余裕あり(10万以上) | +1点 / 超過時 -2点 |
| フォールバック(タグ未設定時) | 最大+5点 |

- 既提案要員は半透明+「提案済」表示、登録ボタン非表示
- 必須スキルカバー率を%表示（色分け）
- 単価差額・経験年数不足を警告表示

### 類似スキルグループ（36グループ定義）
Java≒Kotlin, JavaScript≒TypeScript, AWS, Azure, GCP, Docker≒k8s, PM/PL/SE 等

### データインポート（4方式）
- Excel (.xlsx/.xls) ドラッグ&ドロップ
- CSV (.csv/.tsv) ドラッグ&ドロップ
- JSON テキスト貼り付け（テンプレートコピー機能付き）
- テキスト貼り付け（タブ/カンマ区切り）
- 自動カラムマッピング（完全一致）
- プレビュー・追加/置換モード

## データモデル

### projects (29フィールド)
id, status, shareable, share_note, added_date, source, project_name_original, project_name_rewrite, client_price, purchase_price, **purchase_price_num**, role, location, work_style, period, headcount, required_skills, preferred_skills, **required_skill_tags**, **preferred_skill_tags**, **industry_tags**, **required_experience_years**, description_original, description_rewrite, age_limit, nationality, english, commercial_flow, interview_count

### members (21フィールド)
id, process, affiliation, full_name, initial, contract_employee, desired_price, **desired_price_num**, contact, desired_position, skill_sheet_url, proposal_text, sales_comment, skills_summary, **skill_tags**, **industry_tags**, **experience_years**, experience_summary, nearest_station, available_date, work_preference

### matchings (7フィールド)
id, project_id, member_id, status, note, proposed_date, interview_date

## 技術スタック
- HTML5 / CSS3 / Vanilla JavaScript
- SheetJS (xlsx) — Excelインポート
- Font Awesome — アイコン
- Google Fonts (Noto Sans JP, Inter)
- RESTful Table API — データ永続化

// ---- API Functions ----
async function fetchTable(table, limit = 500) {
    const res = await fetch(`${API_BASE}/${table}?limit=${limit}&sort=-created_at`);
    const data = await res.json();
    return data.data || [];
}

async function createRecord(table, record) {
    const res = await fetch(`${API_BASE}/${table}`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(record)
    });
    return await res.json();
}

async function updateRecord(table, id, record) {
    const res = await fetch(`${API_BASE}/${table}/${id}`, {
        method: 'PUT',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(record)
    });
    return await res.json();
}

async function deleteRecord(table, id) {
    await fetch(`${API_BASE}/${table}/${id}`, { method: 'DELETE' });
}

// ---- Tag Input Component ----
function initTagInput(hiddenId) {
    const wrap = document.getElementById(hiddenId + '-wrap');
    const tagsEl = document.getElementById(hiddenId + '-tags');
    const input = document.getElementById(hiddenId + '-input');
    const hidden = document.getElementById(hiddenId);
    if (!wrap || !input || !hidden) return;

    function getTags() {
        return hidden.value ? hidden.value.split(',').map(t => t.trim()).filter(Boolean) : [];
    }
    function setTags(tags) {
        hidden.value = tags.join(',');
        renderTags();
    }
    function renderTags() {
        const tags = getTags();
        tagsEl.innerHTML = tags.map((t, i) =>
            `<span class="tag-item">${escHtml(t)}<span class="tag-remove" data-idx="${i}"><i class="fas fa-times"></i></span></span>`
        ).join('');
        tagsEl.querySelectorAll('.tag-remove').forEach(btn => {
            btn.addEventListener('click', (e) => {
                e.stopPropagation();
                const tags = getTags();
                tags.splice(parseInt(btn.dataset.idx), 1);
                setTags(tags);
            });
        });
    }
    input.addEventListener('keydown', (e) => {
        if (e.key === 'Enter' || e.key === ',') {
            e.preventDefault();
            const val = input.value.trim().replace(/,/g, '');
            if (val && !getTags().includes(val)) {
                setTags([...getTags(), val]);
            }
            input.value = '';
        }
        if (e.key === 'Backspace' && !input.value) {
            const tags = getTags();
            if (tags.length) { tags.pop(); setTags(tags); }
        }
    });
    // カンマ区切りペースト対応
    input.addEventListener('paste', (e) => {
        setTimeout(() => {
            const parts = input.value.split(/[,、\/]+/).map(s => s.trim()).filter(Boolean);
            if (parts.length > 1) {
                const existing = getTags();
                const merged = [...existing, ...parts.filter(p => !existing.includes(p))];
                setTags(merged);
                input.value = '';
            }
        }, 0);
    });
    wrap.addEventListener('click', () => input.focus());
    renderTags();
    return { getTags, setTags, renderTags };
}

function initNav() {

}

}

function closeModal(id) {
    document.getElementById(id).classList.remove('show');
}

// ---- Toast ----
function showToast(msg) {
    const t = document.createElement('div');
    t.className = 'toast';
    t.textContent = msg;
    document.body.appendChild(t);
    setTimeout(() => t.remove(), 2500);
}

// ---- Dashboard ----
function renderDashboard() {
    const openProjects = projectsCache.filter(p => p.status === 'Open');
    document.getElementById('stat-projects').textContent = openProjects.length;
    document.getElementById('stat-members').textContent = membersCache.length;
    document.getElementById('stat-matchings').textContent = matchingsCache.length;
    document.getElementById('stat-shareable').textContent = openProjects.filter(p => p.shareable === 'OK').length;

    // Recent projects
    const recentEl = document.getElementById('recent-projects');
    const sorted = [...projectsCache].sort((a, b) => (b.added_date || '').localeCompare(a.added_date || '')).slice(0, 8);
    if (sorted.length === 0) {
        recentEl.innerHTML = '<div class="empty-state"><i class="fas fa-inbox"></i><p>案件データなし</p></div>';
    } else {
        recentEl.innerHTML = sorted.map(p => `
            <div class="recent-item" onclick="showProjectDetail('${p.id}')">
                <span class="recent-item-name">${escHtml(p.project_name_rewrite || p.project_name_original || '-')}</span>
                <span class="recent-item-meta">${escHtml(p.added_date || '-')}</span>
            </div>
        `).join('');
    }

    // Searching members
    const searchingEl = document.getElementById('searching-members');
    const searching = membersCache.filter(m => m.process === '案件検索中');
    if (searching.length === 0) {
        searchingEl.innerHTML = '<div class="empty-state"><i class="fas fa-user-slash"></i><p>検索中の要員なし</p></div>';
    } else {
        searchingEl.innerHTML = searching.map(m => `
            <div class="recent-item" onclick="showMemberDetail('${m.id}')">
                <span class="recent-item-name">${escHtml(m.full_name || m.initial)} - ${escHtml(m.desired_position || '-')}</span>
                <span class="recent-item-meta">${escHtml(m.desired_price || '-')}</span>
            </div>
        `).join('');
    }

    // Activity log
    renderActivityLog();

    // Charts
    renderDashboardCharts();
}

let chartProjectStatus = null;
let chartMatchingStatus = null;

function renderDashboardCharts() {
    // 案件ステータス分布（棒グラフ）
    const statusCounts = { Open: 0, Closed: 0, Hold: 0 };
    projectsCache.forEach(p => {
        if (statusCounts.hasOwnProperty(p.status)) statusCounts[p.status]++;
        else statusCounts[p.status || 'Open']++;
    });

    const ctx1 = document.getElementById('chart-project-status');
    if (chartProjectStatus) chartProjectStatus.destroy();
    chartProjectStatus = new Chart(ctx1, {
        type: 'bar',
        data: {
            labels: Object.keys(statusCounts),
            datasets: [{
                label: '件数',
                data: Object.values(statusCounts),
                backgroundColor: ['#10b981', '#ef4444', '#f59e0b'],
                borderRadius: 6,
                maxBarThickness: 60,
            }]
        },
        options: {
            responsive: true,
            maintainAspectRatio: false,
            plugins: { legend: { display: false } },
            scales: {
                y: { beginAtZero: true, ticks: { stepSize: 1 } }
            }
        }
    });

    // マッチングステータス分布（ドーナツ）
    const matchStatusCounts = {};
    matchingsCache.forEach(mt => {
        matchStatusCounts[mt.status] = (matchStatusCounts[mt.status] || 0) + 1;
    });
    const matchLabels = Object.keys(matchStatusCounts);
    const matchValues = Object.values(matchStatusCounts);
    const matchColors = ['#8b5cf6','#3b82f6','#f59e0b','#ec4899','#10b981','#065f46','#ef4444'];

    const ctx2 = document.getElementById('chart-matching-status');
    if (chartMatchingStatus) chartMatchingStatus.destroy();
    chartMatchingStatus = new Chart(ctx2, {
        type: 'doughnut',
        data: {
            labels: matchLabels,
            datasets: [{
                data: matchValues,
                backgroundColor: matchColors.slice(0, matchLabels.length),
                borderWidth: 2,
                borderColor: '#fff',
            }]
        },
        options: {
            responsive: true,
            maintainAspectRatio: false,
            plugins: {
                legend: { position: 'right', labels: { font: { size: 12 } } }
            }
        }
    });
}

// ---- Projects ----
function renderProjects(filter = {}) {
    const tbody = document.getElementById('projects-tbody');
    let list = [...projectsCache];

    if (filter.search) {
        const q = filter.search.toLowerCase();
        list = list.filter(p =>
            (p.project_name_rewrite || '').toLowerCase().includes(q) ||
            (p.project_name_original || '').toLowerCase().includes(q) ||
            (p.required_skills || '').toLowerCase().includes(q) ||
            (p.location || '').toLowerCase().includes(q) ||
            (p.role || '').toLowerCase().includes(q)
        );
    }
    if (filter.shareable) list = list.filter(p => p.shareable === filter.shareable);
    if (filter.workstyle) list = list.filter(p => (p.work_style || '').includes(filter.workstyle));

    // ソート
    list = sortProjects(list);

    if (list.length === 0) {
        tbody.innerHTML = '<tr><td colspan="12" class="empty-state"><i class="fas fa-inbox"></i><p>該当する案件がありません</p></td></tr>';
        updateBulkBar();
        return;
    }

    tbody.innerHTML = list.map(p => {
        const matchCount = matchingsCache.filter(mt => mt.project_id === p.id).length;
        const checked = selectedProjectIds.has(p.id) ? 'checked' : '';
        return `
        <tr>
            <td class="th-check"><input type="checkbox" class="project-check" data-id="${p.id}" ${checked}></td>
            <td><span class="badge badge-${p.status?.toLowerCase() || 'open'}">${escHtml(p.status)}</span></td>
            <td><span class="badge badge-${p.shareable?.toLowerCase() || 'ok'}">${escHtml(p.shareable)}</span></td>
            <td>${escHtml(p.added_date || '-')}</td>
            <td><a href="#" onclick="showProjectDetail('${p.id}'); return false;" style="color:var(--primary);font-weight:500;">${escHtml(truncate(p.project_name_rewrite || p.project_name_original, 30))}</a></td>
            <td>${escHtml(truncate(p.role, 20))}</td>
            <td>${escHtml(p.location || '-')}</td>
            <td>${escHtml(p.work_style || '-')}</td>
            <td>${escHtml(p.purchase_price || '-')}</td>
            <td>${escHtml(truncate(p.source, 15))}</td>
            <td>${matchCount > 0 ? `<span class="badge badge-proposed">${matchCount}件</span>` : '<span style="color:var(--text-secondary);font-size:12px;">-</span>'}</td>
            <td>
                <button class="btn btn-sm btn-secondary" onclick="editProject('${p.id}')"><i class="fas fa-edit"></i></button>
                <button class="btn btn-sm btn-danger" onclick="deleteProject('${p.id}')"><i class="fas fa-trash"></i></button>
            </td>
        </tr>
    `}).join('');

    // チェックボックスイベント
    tbody.querySelectorAll('.project-check').forEach(cb => {
        cb.addEventListener('change', () => {
            if (cb.checked) selectedProjectIds.add(cb.dataset.id);
            else selectedProjectIds.delete(cb.dataset.id);
            updateBulkBar();
        });
    });
    updateBulkBar();
}

function sortProjects(list) {
    const { field, dir } = projectSort;
    const mult = dir === 'asc' ? 1 : -1;
    return list.sort((a, b) => {
        let va, vb;
        if (field === 'match_count') {
            va = matchingsCache.filter(mt => mt.project_id === a.id).length;
            vb = matchingsCache.filter(mt => mt.project_id === b.id).length;
            return (va - vb) * mult;
        } else if (field === 'purchase_price_num') {
            va = parseFloat(a.purchase_price_num) || 0;
            vb = parseFloat(b.purchase_price_num) || 0;
            return (va - vb) * mult;
        } else if (field === 'project_name') {
            va = (a.project_name_rewrite || a.project_name_original || '').toLowerCase();
            vb = (b.project_name_rewrite || b.project_name_original || '').toLowerCase();
            return va.localeCompare(vb) * mult;
        } else {
            va = (a[field] || '');
            vb = (b[field] || '');
            return va.localeCompare(vb) * mult;
        }
    });
}

function updateBulkBar() {
    const bar = document.getElementById('project-bulk-bar');
    const count = selectedProjectIds.size;
    document.getElementById('project-bulk-count').textContent = count;
    bar.style.display = count > 0 ? 'flex' : 'none';
    document.getElementById('project-check-all').checked =
        count > 0 && count === document.querySelectorAll('.project-check').length;
}

function initBulkActions() {
    // 全選択チェックボックス
    document.getElementById('project-check-all').addEventListener('change', (e) => {
        const checks = document.querySelectorAll('.project-check');
        checks.forEach(cb => {
            cb.checked = e.target.checked;
            if (e.target.checked) selectedProjectIds.add(cb.dataset.id);
            else selectedProjectIds.delete(cb.dataset.id);
        });
        updateBulkBar();
    });

    // 一括適用
    document.getElementById('btn-project-bulk-apply').addEventListener('click', async () => {
        if (selectedProjectIds.size === 0) return;
        const newStatus = document.getElementById('project-bulk-status').value;
        const newShareable = document.getElementById('project-bulk-shareable').value;
        if (!newStatus && !newShareable) {
            showToast('変更内容を選択してください');
            return;
        }
        const btn = document.getElementById('btn-project-bulk-apply');
        btn.disabled = true;
        btn.innerHTML = '<i class="fas fa-spinner fa-spin"></i>';
        try {
            const updates = {};
            if (newStatus) updates.status = newStatus;
            if (newShareable) updates.shareable = newShareable;
            for (const id of selectedProjectIds) {
                await fetch(`${API_BASE}/projects/${id}`, {
                    method: 'PATCH',
                    headers: { 'Content-Type': 'application/json' },
                    body: JSON.stringify(updates)
                });
            }
            showToast(`${selectedProjectIds.size}件を更新しました`);
            logActivity('一括更新', 'projects', '', `${selectedProjectIds.size}件`, `${newStatus ? 'ステータス→'+newStatus : ''} ${newShareable ? '共有→'+newShareable : ''}`.trim());
            selectedProjectIds.clear();
            document.getElementById('project-bulk-status').value = '';
            document.getElementById('project-bulk-shareable').value = '';
            await loadAllData();
            renderProjects(getProjectFilters());
            renderDashboard();
            renderMatchingPage();
        } finally {
            btn.disabled = false;
            btn.innerHTML = '<i class="fas fa-check"></i> 適用';
        }
    });

    // 選択解除
    document.getElementById('btn-project-bulk-cancel').addEventListener('click', () => {
        selectedProjectIds.clear();
        document.querySelectorAll('.project-check').forEach(cb => cb.checked = false);
        document.getElementById('project-check-all').checked = false;
        updateBulkBar();
    });
}

function initSort() {
    document.querySelectorAll('#projects-table .sortable').forEach(th => {
        th.addEventListener('click', () => {
            const field = th.dataset.sort;
            if (projectSort.field === field) {
                projectSort.dir = projectSort.dir === 'asc' ? 'desc' : 'asc';
            } else {
                projectSort.field = field;
                projectSort.dir = 'asc';
            }
            // UI更新
            document.querySelectorAll('#projects-table .sortable').forEach(h => {
                h.classList.remove('sort-asc', 'sort-desc');
                h.querySelector('i').className = 'fas fa-sort';
            });
            th.classList.add(projectSort.dir === 'asc' ? 'sort-asc' : 'sort-desc');
            th.querySelector('i').className = projectSort.dir === 'asc' ? 'fas fa-sort-up' : 'fas fa-sort-down';
            renderProjects(getProjectFilters());
        });
    });
}

function renderProjectMatchings(projectId) {
    const related = matchingsCache.filter(mt => mt.project_id === projectId);
    if (related.length === 0) {
        return `<div class="detail-section"><h4><i class="fas fa-users" style="color:var(--primary);margin-right:4px;font-size:13px;"></i>提案中の要員</h4><p style="font-size:13px;color:var(--text-secondary);">紐付いた要員はありません</p></div>`;
    }
    const rows = related.map(mt => {
        const m = membersCache.find(x => x.id === mt.member_id);
        const badge = getMatchingBadge(mt.status);
        return `<tr>
            <td>${escHtml(m ? (m.full_name || m.initial) : '-')}</td>
            <td><span class="badge ${badge.cls}">${escHtml(mt.status)}</span></td>
            <td>${escHtml(mt.proposed_date || '-')}</td>
            <td>${escHtml(mt.interview_date || '-')}</td>
            <td>${escHtml(truncate(mt.note, 30))}</td>
        </tr>`;
    }).join('');
    return `<div class="detail-section"><h4><i class="fas fa-users" style="color:var(--primary);margin-right:4px;font-size:13px;"></i>提案中の要員（${related.length}件）</h4>
        <table class="data-table"><thead><tr><th>要員名</th><th>ステータス</th><th>提案日</th><th>面談日</th><th>メモ</th></tr></thead><tbody>${rows}</tbody></table></div>`;
}

function editProject(id) {
    const p = projectsCache.find(x => x.id === id);
    if (!p) return;
    document.getElementById('project-form-title').textContent = '案件編集';
    const fields = ['id','status','shareable','share_note','added_date','source','project_name_original','project_name_rewrite',
        'client_price','purchase_price','purchase_price_num','role','location','work_style','period','headcount','required_skills','preferred_skills',
        'required_skill_tags','preferred_skill_tags','industry_tags','required_experience_years',
        'age_limit','nationality','english','commercial_flow','interview_count'];
    fields.forEach(f => {
        const el = document.getElementById('pf-' + f);
        if (el) el.value = p[f] || '';
    });
    // タグUIを再描画
    initTagInput('pf-required_skill_tags');
    initTagInput('pf-preferred_skill_tags');
    initTagInput('pf-industry_tags');
    openModal('modal-project-form');
}

async function deleteProject(id) {
    if (!confirm('この案件を削除しますか？')) return;
    await deleteRecord('projects', id);
    await loadAllData();
    renderProjects(getProjectFilters());
    renderDashboard();
    renderMatchingPage();
    showToast('案件を削除しました');
}

// ---- Members ----
function renderMembers(search = '') {
    const grid = document.getElementById('members-grid');
    let list = [...membersCache];

    if (search) {
        const q = search.toLowerCase();
        list = list.filter(m =>
            (m.full_name || '').toLowerCase().includes(q) ||
            (m.initial || '').toLowerCase().includes(q) ||
            (m.skills_summary || '').toLowerCase().includes(q) ||
            (m.desired_position || '').toLowerCase().includes(q)
        );
    }

    if (list.length === 0) {
        grid.innerHTML = '<div class="empty-state" style="grid-column:1/-1"><i class="fas fa-users-slash"></i><p>該当する要員がいません</p></div>';
        return;
    }

    grid.innerHTML = list.map(m => {
        const skills = m.skill_tags
            ? m.skill_tags.split(',').map(s => s.trim()).filter(Boolean).slice(0, 8)
            : (m.skills_summary || '').split(/[,、\/]/).slice(0, 6).map(s => s.trim()).filter(Boolean);
        const processBadge = getProcessBadge(m.process);
        const relatedMatchings = matchingsCache.filter(mt => mt.member_id === m.id);
        const activeMatchings = relatedMatchings.filter(mt => mt.status !== '見送り');
        const matchingInfo = activeMatchings.map(mt => {
            const p = projectsCache.find(x => x.id === mt.project_id);
            return { name: p ? truncate(p.project_name_rewrite || p.project_name_original, 20) : '不明', status: mt.status };
        });
        return `
            <div class="member-card" onclick="showMemberDetail('${m.id}')">
                <div class="member-card-header">
                    <div>
                        <div class="member-name">${escHtml(m.full_name || m.initial)}</div>
                        <div class="member-initial">${escHtml(m.initial)} | ${escHtml(m.affiliation || '-')}</div>
                    </div>
                    <span class="badge ${processBadge.cls}">${escHtml(m.process)}</span>
                </div>
                <div class="member-meta">
                    <div class="member-meta-item"><i class="fas fa-yen-sign"></i>${escHtml(m.desired_price || '-')}</div>
                    <div class="member-meta-item"><i class="fas fa-map-marker-alt"></i>${escHtml(m.nearest_station || '-')}</div>
                    <div class="member-meta-item"><i class="fas fa-calendar"></i>${escHtml(m.available_date || '-')}</div>
                    <div class="member-meta-item"><i class="fas fa-user-tie"></i>${escHtml(m.desired_position || '-')}</div>
                </div>
                ${matchingInfo.length > 0 ? `<div class="member-matching-tags"><i class="fas fa-briefcase" style="font-size:11px;color:var(--primary);margin-right:4px;"></i>${matchingInfo.map(mi => `<span class="badge ${getMatchingBadge(mi.status).cls}" style="margin:1px 2px;font-size:10px;">${escHtml(mi.name)} (${escHtml(mi.status)})</span>`).join('')}</div>` : ''}
                <div class="member-skills">
                    ${skills.map(s => `<span class="skill-tag">${escHtml(s)}</span>`).join('')}
                </div>
                <div class="member-card-actions">
                    <button class="btn btn-sm btn-secondary" onclick="event.stopPropagation(); editMember('${m.id}')"><i class="fas fa-edit"></i> 編集</button>
                    <button class="btn btn-sm btn-danger" onclick="event.stopPropagation(); deleteMember('${m.id}')"><i class="fas fa-trash"></i></button>
                </div>
            </div>
        `;
    }).join('');
}

function showMemberDetail(id) {

    document.getElementById('modal-member-body').innerHTML = `

// スキルの類似グループを検索
function findSimilarSkills(skill) {
    const norm = skill.toLowerCase();
    for (const group of SKILL_GROUPS) {
        if (group.includes(norm)) {
            return group.filter(s => s !== norm);
        }
    }
    return [];
}

function doMatching(projectId) {
    const project = projectsCache.find(p => p.id === projectId);
    if (!project) return;

    const reqTags = parseTags(project.required_skill_tags);
    const prefTags = parseTags(project.preferred_skill_tags);
    const projIndustryTags = parseTags(project.industry_tags);
    const allProjectTags = [...reqTags, ...prefTags];
    const projPriceNum = parseFloat(project.purchase_price_num) || 0;
    const projExpYears = parseFloat(project.required_experience_years) || 0;

    // 既にマッチング登録済み（見送り以外）
    const existingMemberIds = matchingsCache
        .filter(mt => mt.project_id === projectId && mt.status !== '見送り')
        .map(mt => mt.member_id);

    const scored = membersCache.map(m => {
        const memberTags = parseTags(m.skill_tags);
        const memIndustryTags = parseTags(m.industry_tags);
        const memPriceNum = parseFloat(m.desired_price_num) || 0;
        const memExpYears = parseFloat(m.experience_years) || 0;

        let score = 0;
        const matchedRequired = [];
        const matchedPreferred = [];
        const matchedSimilar = [];
        const matchedIndustry = [];
        const penalties = [];

        // --- 必須スキルタグ完全一致 → 3点 ---
        reqTags.forEach(rt => {
            if (memberTags.some(mt => mt === rt)) {
                score += 3;
                matchedRequired.push(rt);
            } else {
                // 類似スキルチェック → 1.5点
                const similar = findSimilarSkills(rt);
                const found = memberTags.find(mt => similar.includes(mt));
                if (found) {
                    score += 1.5;
                    matchedSimilar.push(`${rt}≒${found}`);
                }
            }
        });

        // --- 尚可スキルタグ完全一致 → 1点 ---
        prefTags.forEach(pt => {
            if (memberTags.some(mt => mt === pt)) {
                score += 1;
                matchedPreferred.push(pt);
            } else {
                const similar = findSimilarSkills(pt);
                const found = memberTags.find(mt => similar.includes(mt));
                if (found) {
                    score += 0.5;
                    matchedSimilar.push(`${pt}≒${found}`);
                }
            }
        });

        // --- 業界タグ一致 → 2点/件 ---
        projIndustryTags.forEach(it => {
            if (memIndustryTags.some(mt => mt === it)) {
                score += 2;
                matchedIndustry.push(it);
            }
        });

        // --- ポジション一致 → 2点 ---
        if (project.role && m.desired_position) {
            const roleNorm = project.role.toLowerCase();
            const posNorm = m.desired_position.toLowerCase();
            if (roleNorm.includes(posNorm) || posNorm.includes(roleNorm)) {
                score += 2;
            }
        }

        // --- 経験年数チェック ---
        if (projExpYears > 0 && memExpYears > 0) {
            if (memExpYears >= projExpYears) {
                score += 1; // 満たしていれば +1
            } else {
                penalties.push(`経験${memExpYears}年 < 必要${projExpYears}年`);
                score -= 1; // 不足は -1
            }
        }

        // --- 単価レンジチェック ---
        let priceNote = '';
        if (projPriceNum > 0 && memPriceNum > 0) {
            const diff = projPriceNum - memPriceNum;
            if (diff >= 10) {
                score += 1; // 余裕あり
                priceNote = `余裕+${diff}万`;
            } else if (diff >= 0) {
                priceNote = `差${diff}万`;
            } else {
                // 希望単価 > 仕入単価 → 減点
                score -= 2;
                penalties.push(`単価超過${Math.abs(diff)}万`);
                priceNote = `超過${Math.abs(diff)}万`;
            }
        }

        // --- タグ未設定フォールバック ---
        if (allProjectTags.length === 0 || memberTags.length === 0) {
            const fallbackScore = calcFallbackScore(project, m);
            score += fallbackScore;
        }

        const isExisting = existingMemberIds.includes(m.id);
        const reqCoverage = reqTags.length > 0
            ? Math.round((matchedRequired.length + matchedSimilar.filter(s => reqTags.some(rt => s.startsWith(rt))).length * 0.5) / reqTags.length * 100)
            : null;

        return { member: m, score, matchedRequired, matchedPreferred, matchedSimilar, matchedIndustry, penalties, priceNote, isExisting, reqCoverage };
    }).filter(x => x.score > 0).sort((a, b) => b.score - a.score);

    const resultsEl = document.getElementById('matching-results');
    if (scored.length === 0) {
        resultsEl.innerHTML = '<div class="empty-state"><i class="fas fa-search"></i><p>マッチする要員が見つかりませんでした</p></div>';
        return;
    }

    resultsEl.innerHTML = scored.map(({ member: m, score, matchedRequired, matchedPreferred, matchedSimilar, matchedIndustry, penalties, priceNote, isExisting, reqCoverage }) => {
        const existingLabel = isExisting ? '<span class="badge badge-interviewing" style="margin-left:6px;font-size:10px;">提案済</span>' : '';
        const coverageLabel = reqCoverage !== null ? `<span style="font-size:11px;color:${reqCoverage >= 70 ? 'var(--success)' : reqCoverage >= 40 ? 'var(--warning)' : 'var(--danger)'};margin-left:6px;">必須${reqCoverage}%</span>` : '';
        const opacity = isExisting ? 'opacity:0.55;' : '';
        const priceLabel = priceNote ? `<span style="font-size:11px;margin-left:6px;color:var(--text-secondary);">💰${priceNote}</span>` : '';
        const penaltyHtml = penalties.length > 0 ? `<div style="font-size:11px;color:var(--danger);margin-top:4px;">⚠ ${penalties.join(' / ')}</div>` : '';
        return `
        <div class="match-candidate" style="${opacity}">
            <div class="match-candidate-header">
                <span class="match-candidate-name">${escHtml(m.full_name || m.initial)} <span style="font-size:11px;color:var(--text-secondary)">(スコア: ${score.toFixed(1)})</span>${coverageLabel}${priceLabel}${existingLabel}</span>
                <span class="match-candidate-price">${escHtml(m.desired_price || '-')}${m.experience_years ? ` / ${m.experience_years}年` : ''}</span>
            </div>
            <div class="match-candidate-skills">
                ${matchedRequired.length > 0 ? '<span style="font-size:11px;font-weight:600;color:var(--success);">必須:</span> ' + matchedRequired.map(t => `<span class="skill-tag" style="background:#d1fae5;color:#065f46;">${escHtml(t)}</span>`).join(' ') + ' ' : ''}
                ${matchedPreferred.length > 0 ? '<span style="font-size:11px;font-weight:600;color:var(--info);">尚可:</span> ' + matchedPreferred.map(t => `<span class="skill-tag" style="background:#dbeafe;color:#1e40af;">${escHtml(t)}</span>`).join(' ') + ' ' : ''}
                ${matchedSimilar.length > 0 ? '<span style="font-size:11px;font-weight:600;color:#7c3aed;">類似:</span> ' + matchedSimilar.map(t => `<span class="skill-tag" style="background:#ede9fe;color:#6b21a8;">${escHtml(t)}</span>`).join(' ') + ' ' : ''}
                ${matchedIndustry.length > 0 ? '<span style="font-size:11px;font-weight:600;color:#92400e;">業界:</span> ' + matchedIndustry.map(t => `<span class="skill-tag" style="background:#fef3c7;color:#92400e;">${escHtml(t)}</span>`).join(' ') + ' ' : ''}
                ${matchedRequired.length === 0 && matchedPreferred.length === 0 && matchedSimilar.length === 0 ? escHtml(truncate(m.skills_summary, 100)) : ''}
            </div>
            ${penaltyHtml}
            <div class="match-candidate-actions">
                ${isExisting ? '' : `<button class="btn btn-sm btn-primary" onclick="openMatchingForm('${projectId}', '${m.id}')"><i class="fas fa-plus"></i> マッチング登録</button>`}
                <button class="btn btn-sm btn-secondary" onclick="showMemberDetail('${m.id}')"><i class="fas fa-eye"></i> 詳細</button>
            </div>
        </div>
    `}).join('');
}

// フォールバック：旧キーワードマッチ（タグ未設定時用）
function calcFallbackScore(project, member) {
    const STOP_WORDS = new Set(['以上','経験','対応','可能','業務','開発','設計','担当','必須','不問','スキル','案件','要員','希望','年齢','歓迎','応募','作業','工程','知識','システム','プロジェクト','環境','求める','構築','管理','運用','保守','テスト','実装','製造','詳細','基本','概要']);
    const projectText = `${project.required_skills} ${project.preferred_skills} ${project.role}`;
    const memberText = `${member.skills_summary} ${member.experience_summary} ${member.desired_position}`;
    const pWords = extractCleanKeywords(projectText, STOP_WORDS);
    const mWords = extractCleanKeywords(memberText, STOP_WORDS);
    let score = 0;
    pWords.forEach(pw => {
        if (mWords.some(mw => mw === pw)) score++;
    });
    return Math.min(score, 5); // フォールバックは最大5点に制限
}

function extractCleanKeywords(text, stopWords) {
    if (!text) return [];
    const cleaned = text.toLowerCase()
        .replace(/[（）()「」【】■●・※\n\r]/g, ' ')
        .replace(/[、。]/g, ' ');
    const words = cleaned.split(/[\s,\/]+/).filter(w => w.length >= 2 && !stopWords.has(w));
    return [...new Set(words)];
}

function parseTags(str) {
    if (!str) return [];
    return str.split(',').map(t => t.trim().toLowerCase()).filter(Boolean);
}

function extractKeywords(text) {
    if (!text) return [];
    const cleaned = text.toLowerCase()
        .replace(/[（）()「」【】■●・※\n\r]/g, ' ')
        .replace(/[、。]/g, ' ');
    const words = cleaned.split(/[\s,\/]+/).filter(w => w.length >= 2);
    return [...new Set(words)];
}

async function quickUpdateMatchingStatus(id, newStatus) {
    try {
        await fetch(`${API_BASE}/matchings/${id}`, {
            method: 'PATCH',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({ status: newStatus })
        });
        // キャッシュ更新
        const mt = matchingsCache.find(x => x.id === id);
        if (mt) mt.status = newStatus;
        showToast(`ステータスを「${newStatus}」に変更しました`);
        logActivity('ステータス変更', 'matchings', id, '', newStatus);
        renderDashboard();
    } catch (e) {
        showToast('更新に失敗しました');
    }
}

function editMatching(id) {
    const mt = matchingsCache.find(x => x.id === id);
    if (!mt) return;
    const p = projectsCache.find(x => x.id === mt.project_id);
    const m = membersCache.find(x => x.id === mt.member_id);
    document.getElementById('mtf-id').value = mt.id;
    document.getElementById('mtf-project_id').value = mt.project_id;
    document.getElementById('mtf-member_id').value = mt.member_id;
    document.getElementById('mtf-project-label').value = p ? (p.project_name_rewrite || p.project_name_original) : '';
    document.getElementById('mtf-member-label').value = m ? (m.full_name || m.initial) : '';
    document.getElementById('mtf-status').value = mt.status || '候補';
    document.getElementById('mtf-proposed_date').value = mt.proposed_date || '';
    document.getElementById('mtf-interview_date').value = mt.interview_date || '';
    document.getElementById('mtf-note').value = mt.note || '';
    openModal('modal-matching-form');
}

// ---- Activity Log ----
let activityCache = [];
let notesCache = [];

async function loadActivityLogs() {
    try {
        const [actRes, noteRes] = await Promise.all([
            fetch(`${API_BASE}/activity_logs?limit=30&sort=-created_at`),
            fetch(`${API_BASE}/notes?limit=500&sort=-created_at`)
        ]);
        const actData = await actRes.json();
        const noteData = await noteRes.json();
        activityCache = actData.data || [];
        notesCache = noteData.data || [];
    } catch (e) {
        activityCache = [];
        notesCache = [];
    }
}

async function logActivity(action, targetTable, targetId, targetName, detail) {
    try {
        const now = new Date();
        const timestamp = `${now.getFullYear()}/${String(now.getMonth()+1).padStart(2,'0')}/${String(now.getDate()).padStart(2,'0')} ${String(now.getHours()).padStart(2,'0')}:${String(now.getMinutes()).padStart(2,'0')}`;
        await createRecord('activity_logs', { action, target_table: targetTable, target_id: targetId, target_name: targetName, detail, timestamp });
    } catch (e) {
        console.error('Activity log error:', e);
    }
}

function renderActivityLog() {
    const el = document.getElementById('activity-log-list');
    if (activityCache.length === 0) {
        el.innerHTML = '<div class="empty-state"><i class="fas fa-stream"></i><p>アクティビティなし</p></div>';
        return;
    }
    const ACTION_ICONS = {
        '案件追加': '📋', '案件更新': '✏️', '案件削除': '🗑️',
        '要員追加': '👤', '要員更新': '✏️', '要員削除': '🗑️',
        'マッチング登録': '🤝', 'マッチング更新': '🔄', 'マッチング削除': '🗑️',
        '一括更新': '⚡', 'ステータス変更': '🔄', 'インポート': '📥', 'エクスポート': '📤',
    };
    el.innerHTML = activityCache.slice(0, 20).map(log => `
        <div class="recent-item" style="cursor:default;">
            <span class="recent-item-name">${ACTION_ICONS[log.action] || '📝'} <strong>${escHtml(log.action)}</strong> ${escHtml(log.target_name || '')} ${log.detail ? `<span style="color:var(--text-secondary);font-size:12px;">- ${escHtml(log.detail)}</span>` : ''}</span>
            <span class="recent-item-meta">${escHtml(log.timestamp || '')}</span>
        </div>
    `).join('');
}

// ---- Notes / Comments ----
function renderNotesSection(targetTable, targetId) {
    const notes = notesCache.filter(n => n.target_table === targetTable && n.target_id === targetId);
    const notesList = notes.length > 0
        ? notes.map(n => `
            <div class="note-item">
                <div class="note-content">${escHtml(n.content)}</div>
                <div class="note-meta">
                    <span>${escHtml(n.timestamp || '')}</span>
                    <button class="btn btn-sm btn-danger" onclick="deleteNote('${n.id}', '${targetTable}', '${targetId}')"><i class="fas fa-trash"></i></button>
                </div>
            </div>
        `).join('')
        : '<p style="font-size:13px;color:var(--text-secondary);">メモなし</p>';

    return `
        <div class="detail-section">
            <h4><i class="fas fa-sticky-note" style="color:var(--primary);margin-right:4px;font-size:13px;"></i>メモ（${notes.length}件）</h4>
            <div class="notes-list">${notesList}</div>
            <div class="note-form">
                <textarea id="note-input-${targetId}" rows="2" placeholder="メモを入力..." style="width:100%;padding:8px 12px;border:1px solid var(--border);border-radius:6px;font-size:13px;font-family:inherit;outline:none;resize:vertical;"></textarea>
                <button class="btn btn-sm btn-primary" onclick="addNote('${targetTable}', '${targetId}')" style="margin-top:6px;"><i class="fas fa-plus"></i> メモ追加</button>
            </div>
        </div>
    `;
}

async function addNote(targetTable, targetId) {
    const input = document.getElementById(`note-input-${targetId}`);
    const content = input.value.trim();
    if (!content) return;
    const now = new Date();
    const timestamp = `${now.getFullYear()}/${String(now.getMonth()+1).padStart(2,'0')}/${String(now.getDate()).padStart(2,'0')} ${String(now.getHours()).padStart(2,'0')}:${String(now.getMinutes()).padStart(2,'0')}`;
    await createRecord('notes', { target_table: targetTable, target_id: targetId, content, timestamp });
    await loadActivityLogs();
    // 詳細モーダルを再描画
    if (targetTable === 'projects') showProjectDetail(targetId);
    else if (targetTable === 'members') showMemberDetail(targetId);
    showToast('メモを追加しました');
}

async function deleteNote(noteId, targetTable, targetId) {
    if (!confirm('このメモを削除しますか？')) return;
    await deleteRecord('notes', noteId);
    await loadActivityLogs();
    if (targetTable === 'projects') showProjectDetail(targetId);
    else if (targetTable === 'members') showMemberDetail(targetId);
    showToast('メモを削除しました');
}

// ---- Helpers ----
function escHtml(str) {
    if (!str) return '';
    return String(str).replace(/&/g,'&amp;').replace(/</g,'&lt;').replace(/>/g,'&gt;').replace(/"/g,'&quot;');
}

function truncate(str, len) {
    if (!str) return '-';
    return str.length > len ? str.substring(0, len) + '...' : str;
}

    // ---- Excel File Handler ----
    function handleExcel(file) {
        const reader = new FileReader();
        reader.onload = (e) => {
            try {
                const wb = XLSX.read(e.target.result, { type: 'array' });
                const ws = wb.Sheets[wb.SheetNames[0]];
                const raw = XLSX.utils.sheet_to_json(ws, { header: 1, defval: '' });
                if (raw.length < 2) {
                    showToast('データが不足しています（ヘッダー+1行以上必要）');
                    return;
                }

                // Detect orientation: check if data is row-based or column-based
                // Heuristic: if the first column has known header names, data might be transposed (Excel format from user)
                const firstCol = raw.map(r => String(r[0] || '').trim());
                const knownHeaders = Object.keys(AUTO_MAP[currentTarget] || {});
                const matchCount = firstCol.filter(v => knownHeaders.includes(v)).length;

                if (matchCount >= 3 && matchCount > raw[0].filter(v => knownHeaders.includes(String(v || '').trim())).length) {
                    // Data is transposed (headers in first column, data in subsequent columns)
                    const headers = firstCol;
                    const rows = [];
                    for (let col = 1; col < raw[0].length; col++) {
                        const row = {};
                        headers.forEach((h, i) => {
                            if (h) row[h] = raw[i] ? String(raw[i][col] || '') : '';
                        });
                        rows.push(row);
                    }
                    parsedHeaders = headers.filter(Boolean);
                    parsedRows = rows;
                } else {
                    // Standard orientation (headers in first row)
                    parsedHeaders = raw[0].map(h => String(h || '').trim());
                    parsedRows = raw.slice(1).map(r => {
                        const obj = {};
                        parsedHeaders.forEach((h, i) => { obj[h] = String(r[i] || ''); });
                        return obj;
                    }).filter(r => Object.values(r).some(v => v.trim()));
                }

                markFileLoaded('drop-zone-excel', file.name);
                autoMap();
                showPreview();
            } catch (err) {
                console.error(err);
                showToast('Excelの読み込みに失敗しました: ' + err.message);
            }
        };
        reader.readAsArrayBuffer(file);
    }

    // ---- CSV File Handler ----
    function handleCSV(file) {
        const reader = new FileReader();
        reader.onload = (e) => {
            try {
                const text = e.target.result;
                parseCSVText(text);
                markFileLoaded('drop-zone-csv', file.name);
                autoMap();
                showPreview();
            } catch (err) {
                showToast('CSVの読み込みに失敗しました');
            }
        };
        reader.readAsText(file, 'UTF-8');
    }

    function parseCSVText(text, delimiter = ',') {
        // Simple CSV parser that handles quoted fields
        const lines = [];
        let current = [];
        let field = '';
        let inQuote = false;

        for (let i = 0; i < text.length; i++) {
            const ch = text[i];
            if (inQuote) {
                if (ch === '"' && text[i+1] === '"') {
                    field += '"';
                    i++;
                } else if (ch === '"') {
                    inQuote = false;
                } else {
                    field += ch;
                }
            } else {
                if (ch === '"') {
                    inQuote = true;
                } else if (ch === delimiter) {
                    current.push(field);
                    field = '';
                } else if (ch === '\n' || (ch === '\r' && text[i+1] === '\n')) {
                    current.push(field);
                    field = '';
                    if (current.some(c => c.trim())) lines.push(current);
                    current = [];
                    if (ch === '\r') i++;
                } else {
                    field += ch;
                }
            }
        }
        current.push(field);
        if (current.some(c => c.trim())) lines.push(current);

        if (lines.length < 2) {
            showToast('データが不足しています');
            return;
        }

        parsedHeaders = lines[0].map(h => h.trim());
        parsedRows = lines.slice(1).map(row => {
            const obj = {};
            parsedHeaders.forEach((h, i) => { obj[h] = (row[i] || '').trim(); });
            return obj;
        }).filter(r => Object.values(r).some(v => v));
    }

    // ---- JSON Template ----
    function copyJsonTemplate() {
        const fields = FIELD_DEFS[currentTarget] || [];
        const sample = {};
        fields.forEach(f => { sample[f.key] = ''; });
        const template = JSON.stringify([sample], null, 2);
        navigator.clipboard.writeText(template).then(() => {
            showToast('テンプレートをクリップボードにコピーしました');
        }).catch(() => {
            // Fallback: テキストエリアに直接挿入
            document.getElementById('json-input').value = template;
            showToast('テンプレートをテキストエリアに挿入しました');
        });
    }

    // ---- JSON Parse ----
    function parseJSON() {
        try {
            const input = document.getElementById('json-input').value.trim();
            const data = JSON.parse(input);
            if (!Array.isArray(data) || data.length === 0) {
                showToast('JSON配列形式で1件以上のデータが必要です');
                return;
            }
            parsedHeaders = [...new Set(data.flatMap(d => Object.keys(d)))];
            parsedRows = data.map(d => {
                const obj = {};
                parsedHeaders.forEach(h => { obj[h] = String(d[h] || ''); });
                return obj;
            });
            autoMap();
            showPreview();
        } catch (err) {
            showToast('JSONの解析に失敗しました: ' + err.message);
        }
    }

    // ---- Text Parse ----
    function parseText() {
        const text = document.getElementById('text-input').value;
        if (!text.trim()) {
            showToast('テキストを入力してください');
            return;
        }
        const delimType = document.getElementById('text-delimiter').value;
        const delim = delimType === 'tab' ? '\t' : ',';
        const hasHeader = document.getElementById('text-has-header').checked;

        const lines = text.split('\n').map(l => l.replace(/\r$/, '')).filter(l => l.trim());
        if (lines.length < (hasHeader ? 2 : 1)) {
            showToast('データが不足しています');
            return;
        }

        const splitLines = lines.map(l => l.split(delim));

        if (hasHeader) {
            parsedHeaders = splitLines[0].map(h => h.trim());
            parsedRows = splitLines.slice(1).map(row => {
                const obj = {};
                parsedHeaders.forEach((h, i) => { obj[h] = (row[i] || '').trim(); });
                return obj;
            }).filter(r => Object.values(r).some(v => v));
        } else {
            parsedHeaders = splitLines[0].map((_, i) => `列${i + 1}`);
            parsedRows = splitLines.map(row => {
                const obj = {};
                parsedHeaders.forEach((h, i) => { obj[h] = (row[i] || '').trim(); });
                return obj;
            }).filter(r => Object.values(r).some(v => v));
        }

        autoMap();
        showPreview();
    }

    // ---- Auto Mapping (完全一致のみ) ----
    function autoMap() {
        columnMapping = {};
        const maps = AUTO_MAP[currentTarget] || {};
        const fields = FIELD_DEFS[currentTarget] || [];
        parsedHeaders.forEach(h => {
            const clean = h.trim();
            // 1. AUTO_MAP辞書で完全一致
            if (maps[clean]) {
                columnMapping[h] = maps[clean];
                return;
            }
            // 2. DBフィールドキー名と完全一致
            const directMatch = fields.find(f => f.key === clean);
            if (directMatch) {
                columnMapping[h] = directMatch.key;
            }
        });
    }

    // ---- Show Preview ----
    function showPreview() {
        const section = document.getElementById('import-preview-section');
        section.style.display = 'block';
        document.getElementById('import-row-count').textContent = parsedRows.length;

        // Render mapping grid
        const fields = FIELD_DEFS[currentTarget] || [];
        const mappingGrid = document.getElementById('mapping-grid');
        mappingGrid.innerHTML = parsedHeaders.map(h => {
            const options = fields.map(f =>
                `<option value="${f.key}" ${columnMapping[h] === f.key ? 'selected' : ''}>${f.label}</option>`
            ).join('');
            return `
                <div class="mapping-item">
                    <div class="mapping-source" title="${escHtml(h)}">
                        <i class="fas fa-file-alt"></i>
                        <span>${escHtml(truncate(h, 20))}</span>
                    </div>
                    <div class="mapping-arrow"><i class="fas fa-arrow-right"></i></div>
                    <select class="mapping-select" data-source="${escHtml(h)}">
                        <option value="">-- スキップ --</option>
                        ${options}
                    </select>
                </div>
            `;
        }).join('');

        // Listen for mapping changes
        mappingGrid.querySelectorAll('.mapping-select').forEach(sel => {
            sel.addEventListener('change', () => {
                const source = sel.dataset.source;
                if (sel.value) {
                    columnMapping[source] = sel.value;
                } else {
                    delete columnMapping[source];
                }
                renderPreviewTable();
            });
        });

        renderPreviewTable();
    }

    function renderPreviewTable() {
        const thead = document.getElementById('import-preview-thead');
        const tbody = document.getElementById('import-preview-tbody');
        const fields = FIELD_DEFS[currentTarget] || [];

        // Get mapped fields
        const mappedFields = [];
        for (const [source, target] of Object.entries(columnMapping)) {
            const fieldDef = fields.find(f => f.key === target);
            if (fieldDef) {
                mappedFields.push({ source, target, label: fieldDef.label });
            }
        }

        if (mappedFields.length === 0) {
            thead.innerHTML = '<tr><th>マッピングされたカラムがありません</th></tr>';
            tbody.innerHTML = '';
            return;
        }

        thead.innerHTML = '<tr>' + mappedFields.map(f => `<th>${escHtml(f.label)}</th>`).join('') + '</tr>';
        tbody.innerHTML = parsedRows.slice(0, 5).map(row => {
            return '<tr>' + mappedFields.map(f => {
                const val = row[f.source] || '';
                return `<td title="${escHtml(val)}">${escHtml(truncate(val, 40))}</td>`;
            }).join('') + '</tr>';
        }).join('');
    }

    // ---- Execute Import ----
    async function executeImport() {
        const fields = FIELD_DEFS[currentTarget] || [];
        const mappedFields = [];
        for (const [source, target] of Object.entries(columnMapping)) {
            if (fields.find(f => f.key === target)) {
                mappedFields.push({ source, target });
            }
        }

        if (mappedFields.length === 0) {
            showToast('カラムマッピングを設定してください');
            return;
        }

        const mode = document.getElementById('import-mode').value;
        const records = parsedRows.map(row => {
            const rec = {};
            mappedFields.forEach(({ source, target }) => {
                rec[target] = row[source] || '';
            });
            return rec;
        }).filter(r => Object.values(r).some(v => v));

        if (records.length === 0) {
            showToast('インポート可能なデータがありません');
            return;
        }

        const btn = document.getElementById('btn-import-execute');
        btn.disabled = true;
        btn.innerHTML = '<i class="fas fa-spinner fa-spin"></i> 処理中...';

        try {
            // Replace mode: delete existing records first
            if (mode === 'replace') {
                const existing = currentTarget === 'projects' ? projectsCache : membersCache;
                for (const rec of existing) {
                    await deleteRecord(currentTarget, rec.id);
                }
            }

            // Batch create records (in chunks to avoid overload)
            const chunkSize = 5;
            let imported = 0;
            for (let i = 0; i < records.length; i += chunkSize) {
                const chunk = records.slice(i, i + chunkSize);
                await Promise.all(chunk.map(r => createRecord(currentTarget, r)));
                imported += chunk.length;
                btn.innerHTML = `<i class="fas fa-spinner fa-spin"></i> ${imported}/${records.length}件...`;
            }

            showToast(`${records.length}件のデータをインポートしました`);
            closeModal('modal-import');

            // Refresh data
            await loadAllData();
            renderDashboard();
            renderProjects(getProjectFilters());
            renderMembers(document.getElementById('member-search').value);
            renderMatchingPage();
        } catch (err) {
            console.error(err);
            showToast('インポート中にエラーが発生しました: ' + err.message);
        } finally {
            btn.disabled = false;
            btn.innerHTML = '<i class="fas fa-database"></i> インポート実行';
        }
    }

    // ---- Helpers ----
    function markFileLoaded(zoneId, filename) {
        const zone = document.getElementById(zoneId);
        zone.classList.add('drop-zone-loaded');
        let nameEl = zone.querySelector('.drop-zone-filename');
        if (!nameEl) {
            nameEl = document.createElement('p');
            nameEl.className = 'drop-zone-filename';
            zone.appendChild(nameEl);
        }
        nameEl.innerHTML = `<i class="fas fa-check-circle"></i> ${escHtml(filename)}`;
    }

    // ---- Drag & Drop ----
    function initDropZone(zoneId, fileInputId, handler) {
        const zone = document.getElementById(zoneId);
        const fileInput = document.getElementById(fileInputId);

        ['dragenter', 'dragover'].forEach(ev => {
            zone.addEventListener(ev, (e) => { e.preventDefault(); zone.classList.add('drop-zone-active'); });
        });
        ['dragleave', 'drop'].forEach(ev => {
            zone.addEventListener(ev, (e) => { e.preventDefault(); zone.classList.remove('drop-zone-active'); });
        });

        zone.addEventListener('drop', (e) => {
            const files = e.dataTransfer.files;
            if (files.length) handler(files[0]);
        });

        fileInput.addEventListener('change', () => {
            if (fileInput.files.length) handler(fileInput.files[0]);
        });
    }

    // ---- Init ----
    function init() {
        initTabs();
        initDropZone('drop-zone-excel', 'file-excel', handleExcel);
        initDropZone('drop-zone-csv', 'file-csv', handleCSV);

        document.getElementById('btn-parse-json').addEventListener('click', parseJSON);
        document.getElementById('btn-json-template').addEventListener('click', copyJsonTemplate);
        document.getElementById('btn-parse-text').addEventListener('click', parseText);
        document.getElementById('btn-import-execute').addEventListener('click', executeImport);
        document.getElementById('btn-import-reset').addEventListener('click', () => {
            document.getElementById('import-preview-section').style.display = 'none';
            parsedHeaders = [];
            parsedRows = [];
            columnMapping = {};
        });

        // Import buttons
        document.getElementById('btn-import-project').addEventListener('click', () => open('projects'));
        document.getElementById('btn-import-member').addEventListener('click', () => open('members'));
    }

    return { init, open };
})();
