# SES営業向けマッチング管理ツール

案件・要員・マッチング・進捗を一元管理するWebアプリ（Electron対応）。

## コマンド

```bash
npm run dev              # 開発サーバー
npm run build            # プロダクションビルド
npm run lint             # ESLint
npx prisma migrate dev   # DBマイグレーション作成・適用
npx prisma generate      # Prismaクライアント生成
npm run build:electron   # Electronビルド
```

テストフレームワークは未導入。

## ワークフロー

- **応答はすべて日本語**で行う
- 開発者は**非エンジニア**。専門用語を使う場合は簡単な補足をつける。技術的な判断はClaudeが行い、選択肢がある場合は平易な言葉で説明して確認をとる
- コード変更後は**毎回 `npm run lint` を実行**して問題がないか確認する
- **コミットは自動で行わない**。作業完了後に変更内容を見せて、OKが出たらコミットする
- 小さい変更はそのまま進める。大きい変更や方針判断が必要な場合は事前に確認をとる
- ビルドやコマンドが失敗したときは、**何が起きたか・どう直すかを分かりやすく説明**する
- 作業完了後は**「何をどう変えたか」を非エンジニアにもわかる言葉で要約**する
- 既存コードのパターンや規約は**コードを読んで判断する**（このファイルに細かい規約は書かない）
